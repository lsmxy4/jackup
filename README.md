# README 작성

---

## 프로젝트 개요

- 만든이유
  - 코난 영화의 **순서**와 무엇이 **언재 개봉**했는지 쉽게 볼수있게 하기위해서 만들었다

---

## 화면 구성

- header main footer로 큰 틀을 나누었다
- 이후 나머지 자잘자잘한 div들을 분간하기위해 **class**사용

---

## HTML 작성 시 신경 쓴 점

- 내가 **이해하기 쉽게** 간단하면서 **다른이들**도 **이해하기 쉽게** 클래스의 이름을 간단하게 씀

---

## CSS 작성 시 신경 쓴 점

- css를 작성하면서 잘 **작동되는지** 매번 **저장후 확인**을 반복했다
- 시트들이 잘 보이게 하기위해 하나의 클래스들간의 간격을 띄움

---

## 구현 중 문제와 해결 

- 각 영화에 대한 **설명**과 **방영시기**를 보여주는 창을 만들때 **와이어 프래임**으로 만든후 그것을 코드로 만들기 힘들었었는데 이부분을 일단 내가 할수 있는대까지 만든후 부족한 부분이나 오류 간소화를 ai가 도와줌

---

## 실습을 통해 확인한 점

- 실제로 코드 로 웹페이지를 만드는것이 어려운건 알고있었지만 실제로 해보니 더어려웠지만 그래도 재미가 있었다.

---

## 개선할 점

- 내가 만든것로만 보면 많이 웹사이트라고 하기에는 너무 부실한 감이 없지않아 있다.
  - 그렇기에 이후에 나중에 내가 이사이트를 다시 리뉴얼 할 때가 온다면 다시 한번 해봐야겠다.
  - 또한 너무 단조롭기에 그것에 대한 대책도 마련해야 겠다.
